rappatic on 11.5.18

Titanium, as of November 5th, will now be essentially ceasing development.
Recently the development of Titanium has slowed considerably, and an update hasn't been released in months. We'll still push occassional updates, especially fixes if bugs have been found. Titanium will always continue to be available, but don't except massive updates.


Introducing the Cobalt font stack

rappatic on 9.5.18

Today we released an addition to Titanium - Cobalt! Cobalt is a font stack containing all 850-ish Google Font API fonts. It takes only a second or two to load and it's easy to use.

You can use Cobalt with or without Titanium. The documentation will show you how to use both.

State of development as of v1.1.0

rappatic on 9.2.18

It's been almost two weeks since v1.1.0 has been released. We've been hard at work on Titanium 1.1.1, which is going to be packed with new features. We plan to include a complete set of navigation bars as stated on our board. This update will also include some JS, making the leap from our original vanilla CSS. We're also bringing back shadows. You can find a beta version here.

v1.1.2 will include a ton of smaller things, such as badges, button groups, and options for opacity.
Later on in the development, we plan to bring about some heavier features that include a lot of Javascript. Examples of this include carousels, so you can place multiple images and flip through them with selector buttons, dropdowns, and forms.
