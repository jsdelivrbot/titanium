Titanium

A simple yet powerful web framework built in vanilla CSS


Installation

Titanium uses RawGit to automatically serve the CSS file for your use. Add the following to the head section of your webpage:

            <link rel="stylesheet" type="text/css" href="https://cdn.rawgit.com/rappatic/titaniumcss/master/titanium.css">

And that's all before you're up and running!


Small & Fast

Titanium is only 8 kilobytes, making it quick to load and very lightweight.

Font Support

Easily use a wide range of fonts that require no extra code to access.

In-Depth Tutorials

Learn to use Titanium quickly and easily with our tutorials and expansive documentation.

Responsive

Websites built in Titanium are 100% responsive, and look great on every device.

Automatic Updates

Updates come out every week, and your site will take advantage of them automatically.

Feature-Packed

Titanium is full of powerful features that can be accessed quickly and easily.
